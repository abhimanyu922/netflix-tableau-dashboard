This dashboard provides a visual analysis of Netflix content data. The goal is to understand trends such as content growth over the years, movie vs. TV show distribution, and country-level contributions using interactive visualizations.

## 2. Dataset
The dataset is based on publicly available Netflix data.

**Columns include:**
- Show ID
- Title
- Director
- Cast
- Country
- Release Year
- Date Added
- Rating
- Duration
- Listed In
- Description

## 3. Tools & Technologies
- Tableau Public – Dashboard creation
- Excel / CSV – Data cleaning
- GitHub – Project hosting
- Data Visualization Concepts – Filters, charts, KPIs

## 4. Dashboard Features
- Global content distribution
- Movies vs TV Shows comparison
- Year-wise trend analysis
- Country-wise production insights
- Rating categories and content type mapping
- Fully interactive filters
